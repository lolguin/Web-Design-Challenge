Summary: Latitude vs Various Elements


In this project, we analyzed the impact of different weather elements based on the latitude position. We utilized the OpenWeatherMap API to gather a dataset on over 500 cities.

After gathering and cleaning the data, we utilized Matplotlib to create different scatter plots of the different weather elements vs latitude. The weather elements we analyzed include the following: temperature, cloudiness, wind speed, and humidity. This site provides the various scatter plots along with descriptions of the trends that were observed. You can also access the raw data that was used for this analysis.
